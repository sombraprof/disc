A Decisão Simples: if

Até agora, os nossos programas eram como uma estrada reta: executavam uma instrução após a outra, sem desvios. A estrutura if é o nosso primeiro "desvio condicional". Ela permite que um bloco de código seja executado **apenas se** uma determinada condição for verdadeira.

Exemplo: Verificação de Maioridade

Vamos criar um programa que lê a idade de uma pessoa e exibe uma mensagem apenas se ela for maior de idade.

#include <stdio.h>

int main() {
    // Input variable
    int idade;

    // 1. INPUT
    printf("Digite a sua idade: ");
    scanf("%d", &idade);

    // 2. PROCESSING & DECISION
    // The code inside the braces only runs if 'idade >= 18' is true
    if (idade >= 18) {
        // 3. OUTPUT (Conditional)
        printf("Voce e maior de idade.\n");
    }

    printf("Fim do programa.\n"); // This line always runs

    return 0;
}

A Escolha Dupla: if-else

E se quisermos executar uma ação diferente caso a condição seja falsa? Para isso, usamos a estrutura if-else. Ela cria uma "bifurcação na estrada": se a condição for verdadeira, o programa segue por um caminho; senão (else), ele segue pelo outro.

Exemplo: Par ou Ímpar

Este programa lê um número e informa se ele é par ou ímpar. Só existem estas duas possibilidades.

#include <stdio.h>

int main() {
    int numero;

    printf("Digite um numero inteiro: ");
    scanf("%d", &numero);

    // The condition is if the remainder of the division by 2 is 0
    if (numero % 2 == 0) {
        // Path 1: executed if the condition is true
        printf("O numero %d e PAR.\n", numero);
    } else {
        // Path 2: executed if the condition is false
        printf("O numero %d e IMPAR.\n", numero);
    }

    return 0;
}

Múltiplas Escolhas: else if

Quando temos mais de duas possibilidades, podemos "aninhar" os ifs ou, de forma mais limpa, usar a estrutura else if. Ela permite-nos testar uma série de condições em sequência até que uma delas seja verdadeira.

Exemplo: Classificação de Média

Vamos classificar a situação de um aluno com base na sua média: Aprovado (>= 7.0), Recuperação (>= 5.0) ou Reprovado.

#include <stdio.h>

int main() {
    float media;

    printf("Digite a media do aluno: ");
    scanf("%f", &media);

    if (media >= 7.0) {
        printf("Situacao: APROVADO.\n");
    } else if (media >= 5.0) { // Only checked if the first 'if' was false
        printf("Situacao: RECUPERACAO.\n");
    } else { // Executed if none of the above conditions were true
        printf("Situacao: REPROVADO.\n");
    }

    return 0;
}
